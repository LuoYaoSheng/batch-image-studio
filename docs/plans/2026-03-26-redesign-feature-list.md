# Batch Image Studio - 功能点清单

> **项目定位**：通用化批量图片水印清理工具
> **设计目标**：从豆包专用 → 可自定义规则的通用工具

---

## 一、核心功能概述

### 1.1 应用定位
- **当前状态**：专门针对豆包AI生成图片的水印清理
- **目标状态**：通用化图片区域清理工具，支持用户自定义水印规则
- **核心理念**：规则可配置、处理可预览、批量可执行

### 1.2 技术栈
- **前端**：React 19 + TypeScript + Tailwind CSS + Zustand
- **后端**：Rust + Tauri 2.x
- **AI引擎**：LaMa ONNX 模型（本地运行）
- **跨平台**：macOS / Windows / Linux

### 2.2 规则系统（核心改造）

#### 2.2.1 预设规则库

| 规则名称 | 适用场景 | 区域计算方式 |
|----------|----------|--------------|
| 豆包文字角标 | 豆包AI生成的带文字水印图片 | 右下角，按短边比例计算 |
| 豆包卡片角标 | 豆包AI生成的卡片式水印 | 右下角，稍大尺寸 |
| 浅底快速盖除 | 浅色背景的小图标/文字 | 右下角固定区域 |
| 自定义规则 | 用户自行定义 | 完全自定义 |

#### 2.2.2 自定义规则配置

**规则定义参数**：
```typescript
type WatermarkRule = {
  id: string;
  name: string;                    // 规则名称
  description?: string;            // 规则描述
  region: {
    position: "topLeft" | "topRight" | "bottomLeft" | "bottomRight" | "center" | "custom";
    anchor: { x: number; y: number };  // 0-1，相对坐标
    size: { width: number; height: number };  // 0-1，相对尺寸
    sizingMode: "relative" | "absolute" | "bottomRight";
    baseSize?: number;             // 绝对尺寸时的基准像素
  };
  recommendedMethod: CleanupMethod;  // 推荐的清理方法
  autoApply: boolean;              // 是否自动应用
};
```

**规则管理功能**：
| 功能 | 描述 | 优先级 |
|------|------|--------|
| 规则列表 | 显示所有可用规则 | P0 |
| 规则创建 | 新建自定义规则 | P0 |
| 规则编辑 | 修改现有规则参数 | P0 |
| 规则删除 | 删除自定义规则（预设规则不可删） | P1 |
| 规则启用/禁用 | 控制规则是否在自动检测中生效 | P1 |
| 规则导入/导出 | 分享规则配置 | P2 |

#### 2.2.3 自动规则识别（未来功能）

### 2.4 处理模式

#### 2.4.1 清理方法

| 方法 | 描述 | 适用场景 | 优先级 |
|------|------|----------|--------|
| AI 修复 (inpaint) | 使用 LaMa 模型智能修复 | 复杂背景、大区域水印 | P0 |
| 模糊 (blur) | 高斯模糊处理 | 纯色/简单背景小水印 | P0 |
| 填充 (fill) | 纯色填充 | 纯色背景 | P1 |
| 裁切 (crop) | 裁掉水印区域 | 边缘水印 | P1 |

#### 2.4.2 AI 修复参数
| 参数 | 描述 | 可选值 |
|------|------|--------|
| 模型版本 | LaMa 不同版本 | v1 / v2 / ... |
| 处理质量 | 速度/质量平衡 | 快速 / 平衡 / 精细 |

#### 2.4.3 模糊参数
| 参数 | 描述 | 范围 |
|------|------|------|
| 模糊强度 | 高斯模糊 sigma 值 | 1-50 |

#### 2.4.4 填充参数
| 参数 | 描述 | 可选值 |
|------|------|--------|
| 填充颜色 | 填充用颜色 | 颜色选择器 / 吸管工具 |

## 四、交互流程

### 4.1 典型使用流程

```
1. 用户导入图片（拖拽/选择文件）
   ↓
2. 应用自动识别并应用默认规则
   ↓
3. 用户调整处理区域（拖拽/数值）
   ↓
4. 用户选择处理方法并调整参数
   ↓
5. 点击预览查看效果
   ↓
6. 满意后点击"开始批量处理"
   ↓
7. 等待处理完成，查看统计
   ↓
8. 打开输出文件夹查看结果
```

### 4.2 自定义规则创建流程

```
1. 点击"自定义规则"
   ↓
2. 填写规则名称和描述
   ↓
3. 设置区域参数（位置/尺寸）
   ↓
4. 选择推荐处理方法
   ↓
5. 保存规则
   ↓
6. 规则出现在规则库中供使用
```

## 五、数据结构设计

### 5.1 规则配置
```typescript
type WatermarkRule = {
  id: string;
  name: string;
  description?: string;
  isBuiltIn: boolean;           // 是否内置规则
  regionConfig: {
    position: RegionPosition;
    anchor: Point;
    size: Size;
    sizingMode: SizeHandlingMode;
  };
  recommendedMethod: CleanupMethod;
  defaultParams: {
    blurSigma?: number;
    fillColor?: string;
  };
};

type RegionPosition =
  | "topLeft"
  | "topRight"
  | "bottomLeft"
  | "bottomRight"
  | "center"
  | "custom";

type Point = { x: number; y: number };  // 0-1 相对坐标
type Size = { width: number; height: number };  // 0-1 相对尺寸
```

type ProcessConfig = {

## 七、技术约束
